Upload CSV Manifest

Upload your CSV file to populate or update the product database.

CSV File Format:

Your CSV file should contain the following columns (header names are important):

  • X-Z ASIN (This will be used as the LPN)
  • B00 Asin (Used for ASIN)
  • Fn Sku (Used for FNSKU)
  • Description (Used for Title)
  • MSRP (Optional, used for MSRP)

Other columns will be ignored. If an LPN already exists, its details (ASIN, FNSKU, Title, MSRP) will be updated. New LPNs will be added.
